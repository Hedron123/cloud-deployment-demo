Cloud VM WebApp — Deployment & Hardening

Simple college project demo — Amazon Linux EC2 + Nginx

Project Summary

This web page demonstrates a basic cloud deployment: a static site hosted on an Amazon Linux EC2 instance running Nginx. Minimal host hardening was applied (firewall, SSH restrictions, automatic updates). This page is the project's front-end placeholder and proof of deployment.

How to reproduce (commands used)

Run these on the EC2 terminal as ec2-user (use sudo where shown)

# update packages

Project Files

  • index.html — this front-end file
  • report/project-report.md — assignment report (create locally)

What I did to harden

  • Enabled firewall (open only HTTP/HTTPS/SSH)
  • Disabled root SSH login and password authentication
  • Enabled automatic security updates
  • Installed fail2ban (optional) to block brute force
